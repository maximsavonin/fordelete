               КОМАНДЫ ОТЛАДЧИКА  DEBUG

     Синтаксис вызова:

      DEBUG [[диск:] [путь] имя_файла [параметры_задачи]]

         [[диск:] [путь] имя-файла   --  имя тестируемого файла
         параметры задачи -- параметры, передаваемые в тестиру-
                             емый файл

 ───────────────────────┬──────────────────────────────────────
          Команда       │     Содержание команды
 ───────────────────────┼──────────────────────────────────────
  A [адрес]             │ Ассемблировать. Ввести мнемонику ассемб-
                        │ лера. Нажать "Enter" на пустой строке,
                        │ чтобы закончить ввод (поддержка кодов
                        │ операций 8086/8087)
                        │
  C диапазон  адрес     │ Сравнивает блоки памяти
                        │
  D [адрес]             │ Выводит дамп памяти в HEX- и символьном
                        │ форматах
                        │
  E адрес [байтов]      │ Вводит или изменяет байты и строки в память
                        │
  F диапазон  байты     │ Заполняет память по заданному образцу байтов
                        │
  G [=адрес] [[адрес]..]│ Выполнить.  Выполняет код  в  CS:IP  (или по
                        │ указанному адресу) с возможными контрольными
                        │ остановами в указанных адресах
                        │
  H значение1 значение2 │ Шестнадцатиричная арифметика. Отображает
                        │ (значение1 +  значение2) и (значение1 - зна-
                        │ чение2)
                        │
  I порт                │ Ввод. Отображает значение, считанное из ука-
                        │ занного порта

  L [адрес]             │ Загрузка. Загружает файл по указанному адресу
                        │ или по адресу CS:100h
                        │
  L [адрес] диск нач_сек│ Загружает число_сек абсолютных секторов ука-
    число_сек           │ занного диска, начиная с указанного сектора,
                        │ по указанному адресу или по ажресу CS:100h
                        │
  M  диапазон  адрес    │ Скопировать блок памяти в диапазоне по ука-
                        │ занному адресу
                        │
  N [диск] путь [пара-  │ Имя. Устанавливает имя файла и параметры ко-
                 метры] │ мандной строки
                        │
  O  порт  байт         │ Выводит байт в указанный порт
                        │
  P [адрес][число_шагов]│ Выполнить. Исключить пошаговое выполнение вы-
                        │ зова процедур, циклов, прерываний и т.д.
                        │
  Q                     │ Выход. Завершение работы с отладчиком
                        │
  R [регистр]           │ Регистры. Ввод или отображение значений в ре-
                        │ гистрах/флагах. Устанавливает также для диз-
                        │ ассемблирования указатель   CS:IP   (текущий
                        │ счетчик команд)
                        │
  S  диапазон  байты    │ Поиск в памяти. Показывает адрес, по которому
                        │ содержатся указанные байты или строка
                        │
  T [=адрес] [число_    │ Трассировка. Пошаговое выполнение каждой
                шагов]  │ инструкции
                        │
  U [адрес] или [диапа- │ Дизассемблировать. Отображает мнемонику кода
                   зон] │ операции по указанному адресу или 128 байт,
                        │ начинающиеся с последнего адреса после преды-
                        │ дущего дизассемблирования
                        │
  W [адрес]             │ Записать фрагмент памяти в файл с именем, вве-
                        │ денном в предыдущей команде NAME. Запись на-
                        │ чинается по  указанному адресу или по указа-
                        │ телю CS:100h. Размер записываемого фрагмента
                        │ содержится в BX:CX
                        │
  W [адрес] диск нач_сек│ Записывает число_сек абсолютных секторов дис-
              число_сек │ ка из фрагмента памяти, начинающегося по ука-
                        │ занному адресу или по указателю CS:100h. За-
                        │ пись начинается с сектора  нач_сек  указанного
                        │ диска
                        │
  XA число              │ X-размещение. Размещает указанное число 16Кб
                        │ страниц расширенной памяти
                        │
  XD индекс             │ X-освобождение EMS страниц по указанному индексу
                        │
  XM лог_стр физ_стр    │ X-перевод.  Перевести логические страницы  расши-
             индекс     │ ренной памяти в физические страницы по указанно-
                        │ му индексу
                        │
  XS                    │ X-состояние. Отображает состояние расширенной
                        │ памяти
                        │
  ?                     │ Помощь. Выводит список команд отладчика
                        │
